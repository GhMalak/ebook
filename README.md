# Site estático

O site tem apenas:

- **index.html** – página única (abrir no navegador)
- **css/style.css** – estilos

## Como alterar

- **Razão social, CNPJ, contato, textos:** edite o `index.html` (há comentários `<!-- ALTERE: ... -->` nos trechos que você deve ajustar).
- **Cores, fontes, espaçamentos:** edite o `css/style.css`.

Para publicar, envie a pasta com `index.html` e `css/` para qualquer hospedagem de arquivos estáticos (Vercel, Netlify, GitHub Pages, etc.) ou servidor web.
